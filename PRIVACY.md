# Privacy

AdPages Ghost Newsletter Landing Kit is a local-first resource kit. It is not a Ghost app, does not run a hosted service, and does not make network calls.

## Data handled

The sample files contain example campaign names, UTM values, Open Graph fields, and checklist rows. When you edit those files, the CLI reads them from your local filesystem and writes generated files to the output folder you choose.

## Credentials

This kit does not require Ghost credentials, Ghost Admin API keys, Content API keys, OAuth tokens, cookies, or database access.

## Tracking

The snippets and generator do not add analytics scripts, pixels, hidden form fields, or hidden backlinks. The kit does not add tracking beacons. Generated campaign URLs include visible UTM query parameters only.

## Storage

No data leaves your machine through this kit. Generated files are plain JSON, CSV, Markdown, and Handlebars snippets.
